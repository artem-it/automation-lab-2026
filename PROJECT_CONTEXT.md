# PROJECT_CONTEXT – Automation Lab 2026

## Цель проекта
- Освоить Python и инструменты автоматизации (парсинг, боты, API)
- Создать проекты для портфолио и первых клиентов
- Выйти на первый доход с фриланс-заказов или мини-продуктов

## Текущий статус
- Проектная структура создана (`src/`, `tests/`, `venv/`, `requirements.txt`)
- Git репозиторий и базовые файлы (`README.md`, `.gitignore`, документация) готовы
- Виртуальное окружение активировано, зависимости ещё нужно установить

## Ключевые технологии
- Python 3.12
- Pandas, NumPy
- FastAPI, aiogram
- Requests, aiohttp, AsyncIO
- Git/GitHub

## Рабочие процессы
1. Работа идёт в изолированном venv для проекта
2. Все коммиты фиксируются в Git с понятными сообщениями
3. План на неделю фиксируется в `ROADMAP.md`
4. Все задачи и прогресс фиксируются в `TODO.md`

- Все изменения коммитим регулярно
- TODO.md обновляется каждый день по факту выполненных задач
- ROADMAP.md пересматривается раз в неделю и корректируется
